# EMS AI Agent

新能源平台智能助手，基于 FastAPI + vLLM 构建的轻量级 AI Agent。

## 功能特性

- ✅ 意图识别：自动识别用户查询意图（设备数据、告警、能耗、图表）
- ✅ 设备查询：查询逆变器、电表等设备实时/历史数据
- ✅ 告警管理：查询设备告警信息
- ✅ 图表生成：自动生成前端可渲染的图表配置
- ✅ 上下文支持：单轮对话但保留会话历史
- ✅ 结构化日志：完整的链路追踪
- ✅ 缓存优化：Redis 缓存会话和数据

## 技术栈

- **Web框架**: FastAPI (异步高性能)
- **AI模型**: vLLM (本地部署的大语言模型)
- **数据库**: PostgreSQL (持久化存储)
- **缓存**: Redis (会话和临时数据)
- **日志**: structlog (结构化日志)
